# 🏔 Dolomites Hiking Itinerary Generator

A terminal-based Python program that suggests hiking itineraries in the Dolomites based on your starting area, difficulty level and available time.

## What it does

You answer three questions:
1. **Which area** do you want to hike in (11 zones available)
2. **Difficulty level** — easy, moderate or difficult
3. **How many hours** you have available

The program then suggests a matching itinerary with waypoints, elevation gain, a description and practical tips.

## Areas covered

- Val Gardena
- Val Badia
- Val di Fassa
- Tre Cime di Lavaredo
- Val Pusteria
- Marmolada
- Pale di San Martino
- Cortina d'Ampezzo
- Dolomiti di Brenta
- Agordino e Civetta
- Val di Zoldo

## Requirements

Python 3.6 or higher. No external libraries needed.

## How to run

```bash
python3 dolomites_itinerary.py
```

Press **ENTER** at the end to generate another itinerary, or **Ctrl+C** to quit.

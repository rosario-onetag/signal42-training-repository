## About

EspañolFlow is an interactive Spanish grammar learning platform covering four CEFR proficiency levels: A1 (Beginner) through B2 (Upper Intermediate). The platform features 40 structured grammar lessons with explanations, examples, and interactive exercises.

**This entire application — every line of code, all lesson content, CSS, documentation, and configuration — was generated by Claude Sonnet 4.6 (claude-sonnet-4-6) using Claude Code CLI, without any manual coding by a human developer.**

## Features

- 📚 40 complete grammar lessons across 4 CEFR levels
- 🎯 3 exercise types: multiple choice, fill-in-the-blank, translation
- 📊 Progress tracking persisted across sessions (localStorage)
- 📱 Responsive design (mobile-friendly)
- ⚡ Instant feedback with explanations for every exercise
- 🏆 Lesson completion scores and achievements
- 🔄 Zero backend — runs entirely in the browser

## CEFR Level Progression

| Level | Name | Description | Lessons |
|-------|------|-------------|---------|
| A1 | Beginner | Alphabet, numbers, basic verbs, articles, present tense | 10 |
| A2 | Elementary | Irregular verbs, past tense, object pronouns, gustar | 10 |
| B1 | Intermediate | Subjunctive, future, conditional, por vs para | 10 |
| B2 | Upper Intermediate | Passive voice, reported speech, si clauses, idioms | 10 |

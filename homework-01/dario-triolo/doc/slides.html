🇪🇸 EspañolFlow — Progress System
8 / 10
Progress & Persistence
How it works
  • Storage key espanol_flow_progress in window.localStorage
  • Tracks completed lessons A de-duplicated set of completed lesson IDs (e.g. "a1-01")
  • Stores exercise scores Per-lesson object with correct, total, and completedAt timestamp
  • Calculates completion Percentage completion per CEFR level + overall progress across all 40 lessons
  • Persists across sessions No expiry — stays until the user clears browser storage
Stored JSON structure
{ "completedLessons": [ "a1-01", "a1-02", "a2-01" ], "exerciseResults": { "a1-01": { "correct": 5, "total": 5, "completedAt": "2024-01-15T..." }, "a1-02": { "correct": 4, "total": 5, "completedAt": "2024-01-15T..." } } }
